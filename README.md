# 🐍 Snake Master

> Um jogo moderno da cobrinha desenvolvido em Python com Kivy.

---

## 📖 Sobre o Projeto

Snake Master é um projeto de estudo de desenvolvimento de jogos utilizando Python.

O objetivo não é apenas criar um jogo da cobrinha, mas desenvolver uma arquitetura profissional, escalável e bem documentada.

O projeto seguirá uma evolução incremental através de Sprints, com foco em:

- Arquitetura de Software
- Clean Code
- SOLID
- Desenvolvimento de Jogos
- Documentação
- Testes
- Boas práticas

---

## Tecnologias

- Python 3.13+
- Kivy
- Pytest
- Black
- isort
